# i18n Agent

This agent is responsible for the `lib/i18n/` directory in the Domino Lwa3rin project.

## Responsibilities
- Managing translations for English, French, and Arabic.
- Supporting RTL (Right-to-Left) layouts for Arabic.
- Providing a centralized way to access localized strings.

## Key Files
- [translations.dart](file:///c:/Users/HP/AndroidStudioProjects/DOMINO---lWA3RIN/lib/i18n/translations.dart): Map of localized strings.

## Guidelines
- Avoid hardcoding strings in UI components.
- Ensure Arabic text is properly rendered and aligned.
- Add new keys to all supported languages simultaneously.
